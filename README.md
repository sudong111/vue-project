# 🚀 Guitar web

Vue.js typeScript, Java Spring 으로 구성된 기타 쇼핑몰입니다.

---

## 🧱 기술 스택

-  [Vue.js](https://vuejs.org/)
-  [TypeScript](https://www.typescriptlang.org/)
-  [Spring](https://https://spring.io/)
-  [Tailwind CSS](https://tailwindcss.com/)
